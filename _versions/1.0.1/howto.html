How to use

First, import the groundlevel-utilities module as a dependency:

  <dependencies>
    <dependency>
      <groupId>com.github.jpmsilva.groundlevel-utilities</groupId>
      <artifactId>groundlevel-utilities</artifactId>
      <version>1.0.1</version>
    </dependency>
  </dependencies>

That’s it.
